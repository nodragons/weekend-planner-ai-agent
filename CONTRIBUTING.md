# Contributing to Weekend Planner AI Agent

Thank you for interest in contributing! Here's how you can help:

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone <your-fork-url>`
3. Create a feature branch: `git checkout -b feature/your-feature-name`

## Development Setup

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## Code Standards

- Use [Black](https://github.com/psf/black) for formatting
- Run `flake8` for linting
- Add tests for new features
- Update documentation with changes

## Reporting Issues

- Check existing issues first
- Provide clear description and reproduction steps
- Include Python version and environment details

## Submitting Changes

1. Ensure your code follows project standards
2. Add tests if applicable
3. Update README if needed
4. Submit a pull request with clear description
5. Respond to review feedback

## Areas for Contribution

- Agent instruction improvements
- Additional activity sources
- Better weather integration
- Localization support
- Test coverage
- Documentation
