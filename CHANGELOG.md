# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2025-11-16

### Added
- Initial release of Weekend Planner AI Agent
- Multi-agent system for activity planning
- Weather-aware activity routing
- Google Search integration for finding activities
- Support for local activities, special events, and home activities
- Gemini 2.5 Flash model integration
- Weather router for intelligent flow control

### Features
- Weather checking agent
- Local activities recommendation agent
- Special events discovery agent
- Home activities planning agent
- Summarizer agent for consolidated recommendations
- Sequential execution of research agents

### Architecture
- Root agent orchestrating all sub-agents
- Sequential agent groups for parallel research
- Tool-based agent calling for flexible routing
